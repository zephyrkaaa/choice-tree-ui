# Choice Tree UI

Расширение для SillyTavern — генерирует варианты ответа прямо в чате. После каждого сообщения персонажа появляется блок с карточками, каждая из которых предлагает реплику в своём стиле. Нажал — текст вставился в поле ввода.

---

## Как выглядит

**Обычный режим** — карточки с текстом, можно раскрыть любую чтобы прочитать полностью

> ![preview](https://iili.io/qqoHTiP.png)

**Компактный режим** — 4 кнопки в ряд, текст скрыт, нажал — вставилось

> ![preview](https://iili.io/qqoHIWB.png)

---

## Установка

`Extensions` → `Install Extension` → https://github.com/zephyrkaaa/choice-tree-ui/ → Enter

---

## Настройки

> ![preview](https://iili.io/qqXlC9R.png)

| Настройка | Что делает |
|---|---|
| Расширение включено | Мастер-выключатель |
| Автогенерация | Карточки появляются сами после каждого ответа персонажа |
| Компактный режим | Только 4 маленьких кнопки без текста |
| Количество вариантов | От 2 до 4 |
| Свой промпт | Заменяет встроенный промпт на твой |
| API настройки | Подключить свой API вместо того что используется в ST |

---

## Промпт

В поле настроек уже стоит дефолтный промпт – его можно редактировать прямо там. Если хочешь вернуть к исходному, нажми «Сбросить».

Если пишешь свой промпт, можешь использовать переменные:

```
{{history}}      — последние 8 сообщений чата
{{user}}         — имя пользователя
{{char}}         — имя персонажа
{{lastMessage}}  — последнее сообщение персонажа
{{count}}        — количество вариантов
```

Промпт написан на английском – модели лучше следуют инструкциям на нем, а стиль ответов подстраивается автоматически под то что видит в истории чата. Так что варианты будут соответствовать твоему пресету без лишних указаний.

---

## Архетипы

Четыре стиля ответа, которые предлагает расширение по умолчанию:

- 💙 **Нежный** — мягко, через действие, акцент на чувствах
- 🧊 **Резкий** — сдержанно, дистанция, холодность как защита
- 🔥 **Дерзкий** — уверенно, берёт инициативу
- 🎲 **Дикий** — ломает ожидания, непредсказуемо

---

## Кастомный API

В настройках можно подключить свой API вместо того что стоит в SillyTavern. Поддерживает любой OpenAI-совместимый формат — OpenAI, OpenRouter, Ollama, LM Studio и другие.

После ввода URL и ключа появится кнопка **↻ Загрузить модели** — нажми и выбери нужную из списка.

---

## Особенности

- Варианты не генерируются вне чата с персонажем
- Карточки закрываются крестиком или при смене чата
- В компактном режиме текст всё равно генерируется — просто нажми кнопку и он вставится в поле ввода
- Длинный текст в карточке обрезается, кликни на карточку чтобы раскрыть и прочитать полностью
