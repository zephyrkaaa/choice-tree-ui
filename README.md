# Choice Tree UI

Расширение анализирует текущий контекст и предлагает несколько подходящих реплик для продолжения диалога.

---

## Как выглядит

**Обычный режим** — кнопки с текстом варианта

> ![preview](https://iili.io/qqoHTiP.png)

**Компактный режим** — 4 маленьких кнопки в ряд, текст скрыт

> ![preview](https://iili.io/qqoHIWB.png)

## Установка

`Extensions` → `Install Extension` → https://github.com/zephyrkaaa/choice-tree-ui/ → Enter

Открываются через панель расширений (`Extensions` → `Choice Tree UI`).

## Свой промпт

Чтобы ответы получались именно такими, как тебе хочется, просто впиши свой промпт в поле настроек. А эти переменные помогут тебе всё настроить:

```
{{history}}      — последние 8 сообщений чата
{{user}}         — имя пользователя
{{char}}         — имя персонажа
{{lastMessage}}  — последнее сообщение персонажа
{{count}}        — количество вариантов
```

Пустое поле означает, что будет использоваться стандартный вариант. Он уже полностью готов к ролеплею, так что если не хочется ничего усложнять, просто оставь всё как есть. 

---

## Архетипы ответов

По умолчанию расширение предлагает четыре пути развития диалога:

- 💙 Мягкость – акцент на чувствах и действиях
- 🧊 Холод – ответить максимально сдержанно
- 🔥 Дерзость – брать инициативу в свои руки и действовать уверенно
- 🎲 Хаос – ломать ожидания и идти по непредсказуемому пути

---

## Особенности работы

Что стоит знать:

- Чтобы расширение начало предлагать варианты, нужно сначала зайти в чат с любым персонажем.
- Кнопки выбора автоматически прячутся, как только ты начинаешь печатать свой ответ в поле ввода.
- В компактном режиме текст вариантов скрыт ради чистоты интерфейса, но он всё равно генерируется – просто нажми на кнопку, и готовый ответ появится в поле ввода.
